# Partner logos

**Use for:** Sliding strip of partner logos below the hero (with an appropriate title above).

## Files to add

Add one image per partner, e.g.:

- `partner-1.png`
- `partner-2.png`
- `partner-3.png`
- …

Or use clear names: `acme.png`, `partner-acme.png`, etc.

**Tips:** Use similar height for all logos (e.g. 32–48px) so the strip looks even. SVG or PNG with transparency work well.

## Reference in code

- First logo: `/images/landing/partners/partner-1.png`
- The carousel will load all images from this folder or use a fixed list of paths in code.
