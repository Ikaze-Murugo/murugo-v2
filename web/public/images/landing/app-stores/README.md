# App store badges

**Use for:** Small “Download on the App Store” and “Get it on Google Play” logos below the hero text.

## Files to add

- **Google Play** – Official or custom “Get it on Google Play” badge.  
  Suggested name: `google-play.png` or `google-play.svg`.  
  Recommended height: ~40–48px when displayed.

- **Apple App Store** – Official or custom “Download on the App Store” badge.  
  Suggested name: `apple-store.png` or `apple-store.svg`.  
  Same height as Google Play for alignment.

## Reference in code

- Google Play: `/images/landing/app-stores/google-play.png`
- Apple Store: `/images/landing/app-stores/apple-store.png`
